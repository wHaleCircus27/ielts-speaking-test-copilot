# 工作项目 UI 设计固化

## 文档目标

本文件只固化 `ielts-speaking-test-copilot` 当前工作项目的 UI 设计规范，用于后续开发保持界面、布局和交互一致。

## 产品界面定位

应用是一个 macOS 风格的本地 IELTS Speaking 工作台。首屏必须直接呈现可操作的批改工作区，而不是介绍页或营销页。

核心界面气质：

- 桌面应用感
- macOS 原生窗口隐喻
- 左侧历史记录管理
- 中央任务输入区
- 右侧批改报告区
- 多主题可切换
- 适合长期批改、复盘、训练的高密度工作界面

## 整体布局

应用采用满屏桌面工作台布局：

- 顶部：macOS 菜单栏，高度 32px。
- 中央：应用窗口居中显示。
- 应用窗口最大宽度约 `72rem`。
- 应用窗口最大高度约 `820px`。
- 应用窗口内部包含标题栏、主体区和底部状态栏。

主体区桌面布局：

- 左侧固定历史记录栏，宽度约 `16rem`。
- 右侧为 Workspace。
- Workspace 内部使用左右双栏：
  - 左栏：作业上传、录制、文本输入。
  - 右栏：批改报告、评分、修正和 transcript。

窄屏布局：

- 历史记录栏隐藏。
- Workspace 单列展示。
- 顶部菜单保持单行，长文本截断。
- 不允许出现横向滚动。

## 顶部菜单栏

顶部菜单栏从左到右包含：

- Apple 图标
- `IELTS Assessor`
- `文件`
- `主题切换`
- 本地账户标识
- Wi-Fi 图标
- 电池图标
- 设置按钮
- 帮助按钮
- 当前时间

菜单栏视觉：

- 高度紧凑。
- 字号约 12px。
- 背景随主题变化。
- 支持轻微 backdrop blur。
- 菜单项 hover 时显示低对比背景。

菜单逻辑：

- `IELTS Assessor` 打开应用菜单。
- `文件` 打开文件操作菜单。
- `主题切换` 打开主题选择菜单。
- 设置按钮打开设置弹窗。
- 帮助按钮打开帮助弹窗。

## 应用窗口

窗口由三部分组成：

- 标题栏
- 主体区
- 底部状态栏

标题栏：

- 高度约 44px。
- 左侧为红黄绿三色窗口控制点。
- 中间显示应用标题：`IELTS Speaking Examiner - 雅思口语提分大师`。
- 标题使用小字号、加粗、低透明度。
- 标题栏背景随主题变化。

窗口控制点逻辑：

- 红色：清空当前会话。
- 黄色：打开设置。
- 绿色：打开帮助。

底部状态栏：

- 高度约 24px。
- 左侧显示本地服务状态。
- 右侧显示当前主题和历史记录数量。
- 字号约 10px。
- 整体低对比度，不抢占主要工作区注意力。

## 左侧历史记录栏

历史栏用于沉淀批改记录。

结构：

- 顶部标题：`历史批改记录`
- 右侧显示记录数量。
- 中间为记录列表。
- 底部固定按钮：`新口语作业批改`

记录项内容：

- 文件类型图标
- 作业标题
- 日期
- Band 分数
- hover 后出现删除按钮

记录项状态：

- 默认状态：低对比文本。
- hover：轻微背景高亮。
- active：左侧强调线 + 主题色背景。

空状态：

- 显示本地磁盘图标。
- 提示暂无历史口语作业。
- 提示开始录音或上传文件以批改。

## Workspace 左栏

左栏标题为：`作业上传及录制`。

标题右侧为模式切换控件：

- `音视频录传`
- `手工手写文本`

模式切换使用 segmented control 视觉：

- 外层为低透明背景。
- 当前选中项使用主题主按钮样式。
- 非选中项保持低透明度，hover 后增强。

公共表单字段：

- 作业标题
- 考试部分
- 题目

作业标题：

- 小号 uppercase label。
- 输入框为低对比背景。
- focus 时显示主题色边框或 ring。

考试部分：

- 下拉选择。
- 选项包括 Part 1、Part 2、Part 3。

题目：

- 单行输入框。
- 默认值可使用 IELTS Speaking 示例题。

## 音视频录传模式

上传区使用虚线框。

视觉要求：

- 圆角矩形。
- 2px dashed border。
- 中央上传图标。
- 主文案：`拖拽音频或视频文件至此处`。
- 辅助文案：`支持 MP3, WAV, M4A, MP4 等格式`。
- 下方按钮：`手动浏览文件`。

拖拽状态：

- 工作区覆盖半透明提示层。
- 显示上传图标和 `释放文件导入到工作区`。
- 边框使用强调色。

已选文件状态：

- 使用绿色低透明背景。
- 左侧显示音频或视频图标。
- 中间显示文件名和大小。
- 右侧提供 `清空` 按钮。

## 手工手写文本模式

文本模式显示大文本输入框。

视觉要求：

- label：`写入您的雅思答题脚本`
- textarea 高度至少 128px。
- 支持多行。
- 不允许 resize 破坏布局。
- 底部显示当前字符数。

提交要求：

- 字符数不足时提交按钮 disabled。
- 未连接服务或未配置 Key 时显示明确提示。

## 主操作按钮

左栏底部固定主按钮：

`开始大模型 AI 听写批改`

状态：

- 默认：主题主按钮。
- loading：图标旋转，文案改为 `考官 AI 精审分析中...`。
- disabled：透明度降低，不可点击。

不同主题按钮：

- Claude：橙色圆角胶囊按钮。
- 动物森友会：绿色按钮，带底部厚边。
- 液态玻璃：紫到青色渐变按钮。

## Workspace 右栏

右栏标题：

`口语听写批改工作区 - {作业标题}`

空状态：

- 中央显示音频图标。
- 标题：`等待上传雅思口语作业`
- 描述当前工作区用途。
- 下方三个小卡片：
  - `1. 听写与时间戳`
  - `2. 考官四分细则`
  - `3. 句级瑕疵修正`

有结果状态：

- 顶部评分摘要卡。
- 中间结果 tabs。
- 下方可滚动报告内容。

评分摘要卡：

- 左侧 Band 方块。
- 显示 Overall Band。
- 右侧显示 `雅思口语专家评分`。
- 显示当前模型名称。

## 报告 Tabs

Tabs 包括：

- 综合批语
- 流利度
- 词汇
- 语法
- 发音
- 病句修正

Tabs 视觉：

- 横向排列。
- 小字号。
- 不换行。
- 当前 tab 使用主题主色背景。
- 非当前 tab hover 时显示低透明背景。

## 综合批语面板

内容顺序：

1. 总评与复盘建议
2. transcript 时间戳列表
3. 考官推荐高分示范回答

时间戳列表：

- 卡片内滚动。
- 每行左侧为时间戳。
- 右侧为文本。
- 点击行可尝试跳转音频位置。

高分示范回答：

- 使用绿色低透明背景。
- 边框低对比。
- 文本使用 italic 或 serif 风格。

## 四项评分面板

四项评分面板用于展示：

- Fluency & Coherence
- Lexical Resource
- Grammatical Range & Accuracy
- Pronunciation

内容结构：

- 顶部为评分项标题和分数 badge。
- 中部为标准评语。
- 底部为两列：
  - 突出亮点
  - 改进方向

亮点卡片：

- 绿色低透明背景。
- 使用列表。

改进卡片：

- 琥珀色低透明背景。
- 使用列表。

## 病句修正面板

每条修正卡片包含：

- 分类 badge。
- 原句。
- 箭头。
- 高分重构句。
- 提分原因。

视觉要求：

- 原句使用红色低透明背景。
- 重构句使用绿色低透明背景。
- 箭头在桌面横向展示，在窄屏可旋转为纵向。
- 提分原因使用小字号说明。

分类 badge：

- 语法偏误
- 词汇升级
- 逻辑断流
- 发音讹误

## 设置弹窗

设置弹窗沿用当前项目设置内容，但外层视觉必须匹配工作台主题。

弹窗要求：

- 居中显示。
- 背景遮罩为黑色半透明。
- 支持 backdrop blur。
- 顶部显示标题 `应用设置`。
- 右侧有保存按钮和关闭按钮。
- 弹窗边框、圆角、阴影随主题变化。

## 帮助弹窗

帮助弹窗用于说明当前工作台使用方式。

要求：

- 居中显示。
- 小尺寸卡片。
- 标题：`雅思口语提分大师批改小手册`。
- 右上角关闭按钮。
- 底部确认按钮。
- 视觉随主题变化。

## 主题规范

当前 UI 固定支持三套主题。

Claude 主题：

- 主背景：近白色。
- 卡片：白色。
- 边框：浅灰。
- 主色：橙色。
- 圆角：约 12px。
- 整体感觉：安静、克制、专业。

动物森友会主题：

- 主背景：浅绿色点阵。
- 卡片：米白。
- 边框：暖色粗边。
- 主色：绿色。
- 按钮：底部厚边，轻微拟物。
- 圆角：更大，约 18px。
- 整体感觉：柔和、自然、轻松。

液态玻璃主题：

- 主背景：深色。
- 卡片：透明玻璃感。
- 边框：白色低透明。
- 主色：紫色到青色渐变。
- 使用 blur 和 glow。
- 整体感觉：暗色、磨砂、未来感。

## 响应式规范

桌面宽度：

- 显示历史栏。
- Workspace 左右双栏。
- 报告 tabs 横向滚动但不换行。

窄屏：

- 隐藏历史栏。
- Workspace 单列。
- 顶部菜单文字截断。
- 表单和报告内容不得横向溢出。

## 验收清单

- 打开应用后直接看到工作台。
- 顶部菜单栏、窗口标题栏和底部状态栏完整。
- 左侧历史记录栏显示记录数量和记录项。
- 点击历史记录可切换当前报告。
- 点击新建批改进入空工作台。
- 左栏可切换音视频录传和手工手写文本。
- 右栏空状态、评分摘要、tabs、综合批语、四项评分、病句修正样式完整。
- 设置弹窗和帮助弹窗视觉与主题一致。
- 三套主题下按钮、卡片、侧栏 active、tabs active 都有明确差异。
- 桌面和窄屏均无横向滚动。
